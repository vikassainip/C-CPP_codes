# C/C++ Learning Workspace

This folder contains C and C++ practice programs, exercises, and small projects.

## Structure

- Practice files and examples organized by topic
- STL, OOP, algorithms, strings, trees, and contest problems
- Simple build-and-run examples for learning

## Notes

- Use your preferred C/C++ compiler to build individual source files.
- To use a specific C++ version (make sure the gcc is latest enough to support that), use the following command: `g++ -std=c++23 source_file.cpp -o source_exe`
